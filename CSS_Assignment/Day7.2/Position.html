CSS Position and Z-Index

1. Position Property

The position property in CSS is used to specify the positioning method of an element. It determines how an element is placed in the document flow and how it interacts with other elements.

Syntax:

position: static | relative | absolute | fixed | sticky;

Types of Position:

Type Description Example
static Default position. Elements appear in normal document flow. position: static;
relative Element is positioned relative to its normal position. position: relative; top: 20px; left: 30px;
absolute Element is positioned relative to its nearest positioned ancestor (not static). position: absolute; top: 50px; right: 20px;
fixed Element is positioned relative to the viewport (stays in same place on scroll). position: fixed; bottom: 10px; right: 10px;
sticky Element toggles between relative and fixed depending on the scroll position. position: sticky; top: 0;

Example:

div {
  position: relative;
  top: 30px;
  left: 20px;
}

Note:

  • Properties like top, right, bottom, and left work only with relative, absolute, fixed, or sticky positions.
  • static positioned elements ignore these offset properties.

2. Z-Index Property

The z-index property controls the stacking order of elements that overlap. Elements with a higher z-index value appear in front of those with lower values.

Syntax:

z-index: number;

Example:

div.box1 {
  position: absolute;
  z-index: 1;
}

div.box2 {
  position: absolute;
  z-index: 5;
}

In this example, .box2 will appear above .box1 because it has a higher z-index value.

Important Points:

  • z-index works only on elements with a position value other than static.
  • Higher z-index values bring elements to the front.
  • Elements with the same stacking context depend on HTML order if z-index is equal.
