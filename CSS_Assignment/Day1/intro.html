CSS Lesson 1: What is CSS?

  • CSS stands for Cascading Style Sheets.
  • is used to style and design web pages created with HTML.
  • HTML is about structure & content, CSS is about presentation & looks.
  • CSS can change colors, fonts, spacing, layout, backgrounds, borders, animations and more.
  • Without CSS → websites look plain.
  • With CSS → websites look beautiful & professional.
