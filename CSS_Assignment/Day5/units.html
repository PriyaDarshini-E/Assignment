CSS Units

  • CSS units is used to measure thr properties like width, height, font-size, margin, padding, etc.
  • Units tell the browser how big or small an element should be.

Types of CSS Units

1.Absolute Units

  • px → pixels (most common fixed unit) 1dot = 1px on screen

2.Relative Units

  • % → relative to parent element size the child size will change.
  • em → relative to the parent font size
  • rem → relative to the root element font size
  • vw → 1% of the viewport width
  • vh → 1% of the viewport height
