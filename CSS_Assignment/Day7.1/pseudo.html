CSS Pseudo-Classes and Pseudo-Elements

Pseudo-Class

A pseudo-class is used to define a special state of an element. It allows you to style elements based on their interaction, position, or other dynamic conditions.

Syntax:

selector:pseudo-class {
  property: value;
}

Examples:

a:hover {
  color: red;
}

input:focus {
  border-color: blue;
}

p:first-child {
  font-weight: bold;
}

Common Pseudo-Classes:

  • :hover - When the mouse pointer is over an element.
  • :active - When an element is being clicked.
  • :focus -When an element (like input) is focused.
  • :visited - Styles visited links.
  • :nth-child(n) - Selects the nth child element.
  • :first-child / :last-child - Selects first or last child.

Pseudo-Element

A pseudo-element is used to style specific parts of an element. It allows you to insert or target sub-parts of an element that are not directly in the HTML structure.

Syntax:

selector::pseudo-element {
  property: value;
}

Examples:

p::first-line {
  color: blue;
  font-weight: bold;
}

p::first-letter {
  font-size: 30px;
  color: red;
}

h1::before {
  content: "★ ";
  color: gold;
}

h1::after {
  content: " ★";
  color: gold;
}

Common Pseudo-Elements:

  • ::before - Inserts content before an element.
  • ::after - Inserts content after an element.
  • ::first-letter - Styles the first letter of text.
  • ::first-line - Styles the first line of text.
  • ::selection - Styles the part of text selected by the user.

Difference Between Pseudo-Class and Pseudo-Element

Aspect Pseudo-Class Pseudo-Element
Definition Defines a special state of an element. Styles a specific part of an element.
Syntax selector:pseudo-class selector::pseudo-element
Example a:hover p::first-line
