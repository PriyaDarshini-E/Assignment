CSS Lesson: Grid Layout

CSS Grid is a powerful 2D layout system that helps arrange items into rows and columns easily.

What is Grid?

  • CSS Grid helps create layouts with rows and columns, so 2D
  • We can use fr-Fractional ratio/unit
  • Use display: grid; to start a grid container.

Common Grid Properties

  • grid-template-columns → defines column sizes
  • grid-template-rows → defines row sizes
  • gap → spacing between grid items
  • grid-column / grid-row → item positioning
  • grid-template-areas → name and arrange areas
  • place-item → to align in oh row and col.
