JavaScript DOM Notes

What is DOM?

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of an HTML document as a tree of objects, where each element (like a tag, attribute, or text) becomes a node in the tree.

Example: document.getElementById("demo").innerHTML = "Hello DOM!";

DOM Syntax

Basic syntax to access or modify HTML elements:

  • document.getElementById("id")
  • document.getElementsByClassName("class")
  • document.getElementsByTagName("tag")
  • document.querySelector("selector")
  • document.querySelectorAll("selector")

Example: document.querySelector("p").style.color = "blue";

DOM Selectors

Selectors help us to find elements in the HTML document.

  • By ID: document.getElementById("heading")
  • By Class: document.getElementsByClassName("para")
  • By Tag: document.getElementsByTagName("div")
  • By CSS Selector: document.querySelector(".box")

DOM Properties

DOM properties are used to get or set values of HTML elements.

  • innerHTML() - Change HTML content
  • innerText() - Change text only
  • style() - Change CSS styles dynamically
  • value() - Get or set input values
  • src() - Change image source
  • href() - Change link URL
  • setAttribute() - Add/Modify
  • remove() - want to remove any element or tag
  • appendChild() - want to add any child for a element
  • createElement() - want to create element or tag

Example: 

document.getElementById("title").style.color = "green";

DOM Events

Events are actions that happen in the browser (like clicks, mouse moves, or key presses). We can use JavaScript to handle them.

  • onclick - When a button or element is clicked
  • onmouseover - When mouse hovers over an element
  • onmouseout - When mouse leaves an element
  • onchange - When value changes (input, select)
  • onkeyup - When a key is released

Example: <button onclick="alert('Hello DOM!')">Click Me</button>
