What is Bootstrap?

Bootstrap is a frontend CSS framework used to build responsive, fast and attractive websites using pre-defined classes like container, row, col, btn, etc.

1. Container Classes

<div class="container">Fixed width</div>
<div class="container-fluid">Full width</div>

2. Grid System + Breakpoints

Used for layout. 12 Columns system.

<div class="row">
  <div class="col-6 col-md-4 bg-primary">A</div>
  <div class="col-6 col-md-8 bg-warning">B</div>
</div>

Breakpoints

sm | md | lg | xl | xxl

3. Spacing (Margin & Padding)

<div class="m-3 p-2 bg-dark text-white">Spacing</div>

4. Colors

<p class="text-danger">Red Text</p>
<div class="bg-success text-white p-2">Green BG</div>

5. Borders

<div class="border border-primary p-3">Border Box</div>

6. Buttons

<button class="btn btn-primary">Primary</button>
<button class="btn btn-outline-dark">Outline</button>

7. Flexbox Utilities

<div class="d-flex justify-content-between">
  <div>A</div>
  <div>B</div>
  <div>C</div>
</div>

8. Position Utilities

<div class="position-relative">
  <div class="position-absolute top-0 end-0 bg-warning">Top Right</div>
</div>

9. Shadow

<div class="shadow p-3">Shadow Box</div>

10. Sizing

<div class="w-50 h-25 bg-info">Width 50%</div>

11. Text Utilities

<p class="text-center">Centered Text</p>
<p class="text-uppercase">uppercase</p>

12. Font Utilities
